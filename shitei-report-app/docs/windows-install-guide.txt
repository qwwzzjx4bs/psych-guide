指定医レポート作成 — Windows版 インストールのしかた
================================================

【インストールのしかた】

1. ダウンロードした ZIP ファイルを右クリック →「すべて展開」
2. 展開されたフォルダの中の「指定医レポート作成_インストーラー.exe」をダブルクリック
3. 青い画面で「Windows によって PC が保護されました」と出た場合
   →「詳細情報」をクリック →「実行」をクリック（初回のみ出ることがあります）
4. インストール画面で「次へ」→「インストール」→「完了」を順にクリック
5. デスクトップにできた「指定医レポート作成」のアイコンをダブルクリックして起動

【アプリの使い方（最初の1回）】

1. 「新規」をクリックして入力を開始
2. 入力が終わったら「保存」で .shitei ファイルとして保存
3. 次回からは「開く」で保存したファイルを読み込めます

【困ったとき】

- インストーラーが起動しない
  → ZIP をもう一度「すべて展開」してから、exe をダブルクリックしてください

- アプリが起動しない
  → PC を再起動してから、デスクトップのアイコンから再度起動してください

- アンインストール（削除）したい
  → Windows の「設定」→「アプリ」→「インストールされているアプリ」
  →「指定医レポート作成」を選んで「アンインストール」
