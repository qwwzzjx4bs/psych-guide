# 模範解答パターン

## 成功する精神科臨床図解の構造

```
1. ヘッダー（医療グラデーション背景）
   └─ 疾患名 + ICD-10コード / DSM-5-TR分類
   └─ タグ：診断基準 / 薬物治療 / 非薬物治療

2. 外来クイックリファレンス（最重要3ポイント）
   └─ 診察中に即座に参照できる要点
   └─ 赤・青・紫の色分け番号付きカード

3. 診断基準セクション
   └─ DSM-5-TR A〜E基準を構造的に表示
   └─ ICD-10コードとの対比
   └─ 重要項目（①②）を赤字で強調

4. 診断ツリー / 鑑別フロー
   └─ 主訴から診断名までのビジュアルフロー
   └─ 除外すべき疾患の色分け分岐

5. サブタイプ・Specifier分類
   └─ 治療方針を変えるものを優先（赤カード）
   └─ 薬剤選択に影響するもの（青カード）

6. 鑑別診断
   └─ 必須除外（器質性・双極性等）
   └─ 鑑別すべき精神疾患

7. 薬物治療テーブル
   └─ ファーストライン → セカンドライン → 増強の順
   └─ 薬剤名・用量・特徴・エビデンスレベルバッジ

8. 非薬物治療
   └─ 心理療法（CBT, IPT, MBCT, SPT）
   └─ 物理的療法（mECT, rTMS）
   └─ 生活指導

9. 禁忌・注意・モニタリング
   └─ 赤ボックス：禁忌
   └─ グレーボックス：モニタリング項目

10. 目次（フローティング）
    └─ デスクトップのみ表示
    └─ 各セクションへのアンカーリンク
```

## 疾患別の重点構成例

### うつ病（MDD）のとき

**特に強調すること**:
- 双極性鑑別（必須除外の筆頭）
- PHQ-9による重症度評価
- 非薬物治療（CBT・支持的療法）を薬物治療と同等の比重で
- 維持療法の期間（初発6〜12ヶ月）

**追加すべきセクション**:
- 自殺リスク評価のフロー
- 休職・復職タイムライン
- 薬物療法の効果判定フロー（4週→8週→治療抵抗性判定）

---

### 統合失調症のとき

**特に強調すること**:
- 陽性症状・陰性症状・認知症状の3分類
- PANSS評価
- 第一世代 vs 第二世代抗精神病薬の選択基準
- 治療抵抗性のクロザピン適応（≥2剤失敗後）

**追加すべきセクション**:
- 再発リスク因子と服薬アドヒアランス強化
- 社会復帰・SST・デイケア
- 家族心理教育

---

### 双極性障害のとき

**特に強調すること**:
- 双極Ⅰ型 vs Ⅱ型 vs 気分循環症の鑑別
- SSRI単剤投与の危険性（躁転・急速交代化）
- 気分安定薬（リチウム・バルプロ酸・ラモトリギン）の使い分け
- 血中濃度モニタリング

**追加すべきセクション**:
- 急性躁エピソード vs 急性うつエピソードの治療分岐
- 維持療法の長期戦略
- 急速交代型への対応

---

### 不安症（GAD / パニック症 / 社交不安症）のとき

**特に強調すること**:
- 各不安症の中核症状の違い
- BZD系抗不安薬の長期使用回避
- SSRIが全ての不安症のファーストライン
- CBTの重要性（薬物療法と同等以上）

**追加すべきセクション**:
- 曝露療法の基本原則
- 身体疾患との鑑別（甲状腺、心疾患等）

## エビデンスレベルバッジの使用ルール

```html
<!-- 全ての治療選択肢にバッジを付ける -->
<span class="badge-level-a">Level A</span>  <!-- 複数RCT・メタ解析で確立 -->
<span class="badge-level-b">Level B</span>  <!-- 単独RCTまたは観察研究 -->
<span class="badge-level-c">Level C</span>  <!-- 専門家コンセンサス・症例報告 -->
```

**Level Aを付けるもの（例）**:
- SSRI/SNRIのうつ病・不安症への適応
- CBTのうつ病・不安症・OCD・PTSDへの適応
- mECTの重症うつ・緊張病への適応
- リチウムの双極性障害への適応

**Level Bを付けるもの（例）**:
- 支持的精神療法（全般）
- 運動療法のうつ病への効果
- ラメルテオンの不眠症への適応

## 品質チェックリスト

作成後、以下を確認：

### 診断精度
- [ ] ICD-10コードが正確に記載されている
- [ ] DSM-5-TR診断基準のA〜E基準が網羅されている
- [ ] 治療方針を変えるSpecifierが全て記載されている
- [ ] 鑑別診断に双極性・器質性が含まれている
- [ ] 重症度評価スケールが記載されている

### 治療内容
- [ ] ファーストライン薬物治療が用量範囲と共に記載されている
- [ ] エビデンスレベルバッジが全治療選択肢に付いている
- [ ] 非薬物治療が薬物治療と同等の比重で記載されている
- [ ] 治療抵抗性への対応が記載されている（mECT, rTMS, クロザピン等）
- [ ] 禁忌・注意事項・モニタリング項目がある

### デザイン
- [ ] Lucide iconのみ使用（絵文字禁止）
- [ ] 緊急度・重要度で色分けされている（赤=緊急、青=標準、紫=増強）
- [ ] 診断ツリーが視覚的フローとして表現されている
- [ ] 外来中に素早く参照できる「クイックリファレンス」がある
- [ ] スマホでも読みやすいレスポンシブ対応
- [ ] 目次がフローティング表示される（デスクトップ）
