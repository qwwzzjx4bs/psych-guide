## DSM-5-TR 共通 Specifier（特定用語）

| 用語 | 意味 | 適用例 |
|------|------|--------|
| 不安を伴う | 緊張・心配・落ち着きのなさを伴う | うつ病、双極性障害 |
| 混合性の特徴を伴う | 反対極の症状が≥3項目存在 | うつ病エピソード、躁エピソード |
| 急速交代型 | 12か月内に≥4エピソード | 双極性障害 |
| 非定型の特徴を伴う | 気分反応性あり + 鉛様麻痺・過眠・過食・拒絶過敏 | うつ病 |
| 憂うつ性の特徴を伴う | 喜び消失・朝方悪化・早朝覚醒・精神運動変化顕著 | うつ病 |
| 緊張病を伴う | 昏迷・常同症・反響動作・拒絶症など | うつ病、統合失調症 |
| 産後発症 | 周産期（産後4週以内）に発症 | うつ病、双極性障害 |
| 季節性の様式 | 特定の季節に規則的に発症・寛解 | うつ病、双極性障害 |
| 精神病性の特徴を伴う | 気分に一致/不一致の幻覚・妄想 | うつ病、双極性障害 |
| 部分寛解 | 基準を完全には満たさないが症状が一部残存 | 全障害群 |
| 完全寛解 | 症状なし（期間は疾患により異なる） | 全障害群 |

## エビデンスレベル定義

| レベル | 定義 | 例 |
|--------|------|---|
| **Level A** | 複数のRCTまたはメタ解析で有効性が確立 | 大うつ病へのSSRI |
| **Level B** | 少なくとも1つのRCTで有効性が示唆、または複数の観察研究 | 支持的精神療法 |
| **Level C** | 専門家コンセンサス・症例報告・臨床経験に基づく | 新規薬剤の適応外使用 |
| **Level D** | エビデンス不十分または相反する結果 | - |
