外来で押さえる4ポイント

診察中に即座に確認すべき要点

1
SGAを第一選択・単剤で十分試行

初回エピソードはSGAを低用量から。十分量で4〜6週試行後に評価。多剤併用は原則避ける

2
代謝・EPS・QTを患者別に評価

若年者は代謝系（オランザピン回避）、高齢者はEPS・鎮静・QT。副作用プロファイルで薬剤選択

3
2剤失敗 → クロザピンを躊躇なく

十分量の異なる2剤で無効なら治療抵抗性。クロザピンは最も強いエビデンスを持つ唯一の選択肢

4
アドヒアランス不良はLAIへ

服薬中断リスクが高い患者は早期にLAIを検討。再発防止の観点から経口剤と同等以上の選択肢

受容体と副作用の対応
  • D2強遮断 → EPS・高プロラクチン血症（FGA・リスペリドン高用量）
  • 5-HT2A拮抗 → D2遮断によるEPSを相殺（非定型性の核心）
  • H1強拮抗 → 鎮静・体重増加（オランザピン・クロザピン）
  • M1拮抗 → 口渇・便秘・認知障害（クロザピン・オランザピン）
  • α1拮抗 → 起立性低血圧・ふらつき（クロルプロマジン・クエチアピン）
部分作動薬（アリピプラゾール・ブレクスピプラゾール）の特性
  • D2部分作動 → ドパミン過活動では拮抗薬として働き、正常〜低活動では作動薬として働く
  • 代謝への影響 → H1・M1親和性が低いため体重増加・代謝異常が少ない
  • プロラクチン → D2完全遮断でないため上昇しにくい
  • アカシジア → 特にアリピプラゾールで比較的多い（ブレクスピプラゾールは少ない傾向）

LAI（持続性注射剤）

Long-Acting Injectable — アドヒアランス改善と再発予防

  • ・服薬アドヒアランス不良の既往 / 中断による再発歴
  • ・患者・家族の希望（注射のほうが管理しやすい場合）
  • ・経口剤での安定後、維持療法として変更
  • ・初回エピソード後でも早期LAI導入を検討（日本ガイドライン2022でも言及）

  • 悪性症候群（NMS）
    高熱・筋強剛・自律神経不安定・意識障害・CK上昇。全AP薬で起こりうる。直ちに投与中止・ICU管理
  • 遅発性ジスキネジア（TD）
    長期使用後に出現する不随意運動（口周囲・体幹）。FGAリスク高。可逆性低い。バルベナジン（有効）
  • QTc延長・心室頻拍
    電解質異常・併用薬に注意。ハロペリドール・クロルプロマジン・クエチアピンで要注意
クロザピン固有の重大副作用
  • 無顆粒球症（1-2%）
    好中球<1500/μL→投与中止。登録制CPMS下での血液検査スケジュール遵守が必須
  • 心筋炎・心筋症
    開始後4-8週に好発。発熱・胸痛・心悸亢進・CRP上昇→心電図・エコーで確認
  • 痙攣発作
    用量依存性（高用量で↑）。EEG異常歴・既往てんかんで慎重
  • 腸閉塞（腸管麻痺）
    強い抗コリン→便秘→腸閉塞。下剤の積極的使用、排便管理
EPS（錐体外路症状）の種類と対応
EPS種類出現時期症状
急性ジストニア1-3日頸部・眼球・体幹の筋収縮
パーキンソニズム数週間振戦・固縮・無動
アカシジア1-6週静座不能・不安感
遅発性ジスキネジア数ヶ月〜年口周囲・体幹の不随意運動
対応・管理
  • ・急性ジストニア → ビペリデン（アキネトン）筋注で即効性
  • ・パーキンソニズム → 減量 or 抗パーキンソン薬（ビペリデン経口）
  • ・アカシジア → 減量 / βブロッカー（プロプラノロール） / BZD / ミルタザピン
  • ・TD → 可能ならSGAへ変更 / クロザピンへ / バルベナジン（VMAT2阻害薬）
  • ・DIEPSS（日本語版副作用評価尺度）で定期評価

評価スケール
  • ・PANSS：陽性/陰性/一般精神病理の3軸
  • ・BPRS：簡易精神症状評価
  • ・DIEPSS：薬原性錐体外路症状（EPS）
  • ・AIMS：不随意運動（TD評価）
  • ・SAS：Simpson-Angus Scale（EPS）
  • ・BAS：Barnes Akathisia Scale
QTc管理
  • ・QTc > 500ms → 投与中止検討
  • ・ベースラインから +60ms → 要注意
  • ・リスク因子：低K・低Mg・徐脈・女性・心疾患
  • ・併用回避：抗不整脈薬、一部抗菌薬（フルオロキノロン系等）
高齢者・認知症
  • ・全AP薬で死亡リスク増のBlack Box Warning
  • ・最小量・短期使用が原則
  • ・転倒・骨折リスク：EPS・鎮静に注意
  • ・抗コリン認知機能↓：クロザピン・オランザピン要注意
  • ・非薬物療法（環境調整・心理ケア）を第一選択

相互作用・禁忌・注意事項

安全処方のための必須知識

主要な禁忌
  • 糖尿病（コントロール不良含む）
    オランザピン・クエチアピン（速放・徐放）の禁忌。血糖値・HbA1c確認必須
  • QTc延長（先天性）
    ハロペリドール、クロルプロマジン、クエチアピン等で要注意
  • 重篤な心疾患
    クロザピンは心筋炎・心筋症既往で慎重
  • Parkinson病・Lewy小体型認知症
    D2拮抗薬でパーキンソン症状著明悪化。クエチアピンが比較的安全
  • アドレナリン投与中
    α1遮断薬（CPZ・クロザピン等）との併用でアドレナリン昇圧反転（血圧低下）
主要な薬物相互作用
CYP1A2（クロザピン・オランザピン）
  • ・喫煙（CYP1A2誘導）→ 濃度低下。禁煙時に濃度上昇・副作用増
  • ・フルボキサミン（CYP1A2阻害）→ 血中濃度上昇
  • ・シプロフロキサシン → 血中濃度上昇
CYP3A4（アリピプラゾール・ブレクスピプラゾール・ルラシドン）
  • ・強阻害薬（ケトコナゾール・クラリスロマイシン）→ 血中濃度著明上昇 → 減量
  • ・強誘導薬（カルバマゼピン・リファンピシン）→ 血中濃度低下 → 増量検討
  • ・ルラシドンはCYP3A4阻害薬との併用禁忌
CYP2D6（リスペリドン・アリピプラゾール等）
  • ・パロキセチン・フルオキセチン（CYP2D6阻害）→ リスペリドン血中濃度上昇
  • ・Poor metabolizer (PM) では通常量で過剰に
QTc延長薬との併用
  • ・抗不整脈薬・三環系抗うつ薬・一部抗菌薬（フルオロキノロン）との累積リスク
  • ・電解質異常（低K・低Mg）の補正が先決
妊婦・授乳婦への対応
  • ・精神症状のコントロールは母児双方にとって重要（未治療のリスクも考慮）
  • ・比較的安全性データが多い：ハロペリドール・クロルプロマジン・リスペリドン
  • ・オランザピンは妊娠糖尿病リスクあり
  • ・分娩直前投与 → 新生児EPS・離脱症状リスク
  • ・授乳：多くが母乳中に移行。リスクベネフィット評価
  • ・クロザピンは授乳中禁忌（母乳中濃度が高い）
  • ・産科・周産期精神医学専門家との連携を強く推奨
