精神保健指定医支援 — 令和7年新基準

指定医レポート作成アプリ

Word/Excelのように保存・上書き・バージョン管理ができるデスクトップ版ウィザード

主な機能

  • 7ステップ × 5症例の入力ウィザード（現行HTML版と同等）
  • .shitei ファイルの新規・開く・保存・上書き保存
  • ドキュメント内蔵バージョン履歴（最大50件）
  • 55項目チェックリストの自動判定 + 手動確認
  • 様式3-1プレビュー・印刷
  • 旧ウィザード JSON のインポート
  • すべてローカル保存（外部送信なし）

インストール・起動

リポジトリ内の shitei-report-app/ からビルドします。

cd shitei-report-app
npm install
npm run tauri dev    # 開発
npm run tauri build  # 配布用ビルド

macOS ビルド成果物: shitei-report-app/src-tauri/target/release/bundle/

旧HTMLウィザードからの移行

  1. 旧 shitei-report-wizard.html で JSON を保存
  2. デスクトップアプリの「旧JSON取込」で読み込み
  3. 「名前を付けて保存」で .shitei として保存
