BFCRSとは

・Bush, Fink, Petrides, Dowling, Francis（1996）が開発した緊張病（カタトニア）の標準的評価尺度

・23項目からなる包括的評価尺度（BFCRS）

・前半14項目はBFCSS（スクリーニング）として単独使用可能

・評価期間：現在の観察（過去数日以内）。評価時間：約10〜20分

・スコア範囲：0〜69点（一部バイナリ項目あり）

BFCSS スクリーニング（項目1〜14）

0 0項目：緊張病の可能性低い
1 1項目：要注意、経過観察
≥2 2項目以上：緊張病疑い → 全23項目評価・ベンゾジアゼピン反応試験を検討

悪性緊張病・鑑別

悪性緊張病の徴候：

項目23（自律神経異常）≥2 + 重篤な運動症状
発熱・頻脈・血圧不安定・発汗
横紋筋融解・腎不全リスク

NMS鑑別：抗精神病薬使用歴、CK上昇、発症経過

ベンゾジアゼピン反応試験（Lorazepam challenge）

実施方法：

ロラゼパム 1〜2mg を静注または筋注。投与前後（30分以内）にBFCRSを評価。50%以上の改善を陽性とする。

判定：

陽性緊張病を支持。継続的なBZP投与を検討
陰性他の緊張病原因・NMS・ECTを検討
BFCSS スクリーニング項目（Bush-Francis Catatonia Screening Scale）― 項目 1〜14 2項目以上陽性 → 緊張病疑い 陽性 0 / 14項目
1
興奮 Excitement
明らかに目的のない極度の多動・運動不穏（アカシジアや目標指向の焦燥とは区別）
－
定義：明らかに目的のない極度の過活動・運動不穏。アカシジア（静坐不能症）や目標指向性の焦燥感とは区別すること。
注意：緊張病性興奮は「無目的」であることが特徴。苦悶・不快感に基づくアカシジアや、妄想・命令幻声に従った行動とは異なる。
観察・評価ポイント：患者が静止できているか、動きに目的性があるか、面接中・安静時の動きを観察。誘導で行動が変わるか確認。
0　なし1　軽度2　中等度3　重度
過活動なし 過剰な運動が見られるが間欠的（断続的に止まる時間がある） 持続的な運動・多動、休止期なし（ひっきりなしに動き続ける） 完全な緊張病性興奮。際限のない激しい運動活動（制御が非常に困難）
2
不動 / 昏迷 Immobility / Stupor
極度の活動低下・不動。刺激への反応が最小限
－
定義：極度の活動低下・不動状態。刺激への反応が最小限となる。
注意：うつ病性制止（抑制）・昏睡・鎮静剤による意識障害と鑑別が必要。緊張病性昏迷は急激な発症や他の緊張病徴候の併存が特徴。
評価方法：声かけへの反応、疼痛刺激（胸骨圧迫等）への反応を観察。眼球の動き・まばたきの有無も確認。
0　なし1　軽度2　中等度3　重度
活動は正常範囲 異常なほど静止している。短時間の交流は可能（呼びかけに応じることがある） 外界との交流がほぼない。刺激には反応するが積極的関与なし 昏迷状態。疼痛刺激にも反応しない
3
緘黙 Mutism
言語的に無反応または最小限の反応
－
定義：言語による反応がないか、最小限の反応のみ。
注意：口腔・喉頭の器質的障害、失語症との鑑別が必要。緊張病性緘黙では発声器官に器質的問題はない。書字・筆談で意思疎通できる場合は軽症。
評価方法：5分間の会話を試み、発話語数を数える。囁き声・口の動きだけでも1とする。
0　なし1　軽度2　中等度3　重度
通常の言語表出 大多数の質問に言語的に反応しない、または聞き取れない囁き声のみ 5分間に20語以下しか話さない 発語が全くない（完全緘黙）
4
凝視 Staring
固定した視線・視覚的走査の減少・瞬目の減少
－
定義：固定した視線、周囲の視覚的走査の減少、瞬目の減少。
注意：正常では通常3〜5秒ごとに視線を変える。瞬目は通常10〜15回/分。緊張病では視線が固定し、まばたきが著しく減少する。
評価方法：視線の変化頻度を計測（20秒以上同一方向なら2以上）。瞬目回数を1分間計測。
0　なし1　軽度2　中等度3　重度
視線・瞬目は正常 アイコンタクト不良。20秒未満ごとに視線を変える、瞬目が減少 視線を20秒以上保持。時折注意を移すが固定傾向あり 完全に固定した視線。無反応（瞬目がほぼない、外的刺激にも視線が動かない）
5
姿勢固持 / 強硬症 （カタレプシー）Posturing / Catalepsy
姿勢の自発的な保持（奇異なものを含む日常的姿勢の長時間保持）
－
定義：自発的に姿勢を保持する。日常的姿勢（座位・立位の長時間保持）から奇異な姿勢まで含む。
注意：蝋屈症（蝋様固縮・項目13）と混同しないこと。ここでは「自発的な姿勢固持（カタレプシー）」の持続時間を評価する。検査者が置いた体位をそのまま保持し続けることは蝋屈症（項目13）。
評価方法：異常な姿勢の持続時間を計測。奇異な姿勢（例：腕を水平に上げたまま、頭を後屈させたまま等）の有無を確認。
0　なし1　軽度2　中等度3　重度
自発的な姿勢保持なし 1分未満の姿勢保持（日常的または奇異な姿勢を短時間保持） 1分以上15分未満の姿勢保持 15分以上の姿勢保持、または奇異な姿勢の保持
6
扮相 Grimacing
奇妙な表情の保持
－
定義：奇妙または不自然な顔の表情の保持。
注意：情動に基づく表情変化（泣き顔・怒り顔）や、痛みに伴う苦悶表情とは区別する。緊張病の扮相は文脈に無関係な固定した奇妙な表情が特徴。
0　なし1　軽度2　中等度3　重度
扮相なし 10秒未満の扮相（断続的） 10秒以上1分未満の持続 奇妙な表情の長時間保持（1分以上）または著しく奇妙な表情
7
反響動作 / 反響言語 Echopraxia / Echolalia
検査者の動作・発言の模倣
－
定義：検査者の動作（エコープラクシア）または発言（エコーラリア）の模倣。
注意：通常の社会的模倣（うなずき等）とは区別する。発言をそのまま繰り返す（オウム返し）、動作をそのままなぞるなどが対象。
誘発方法：患者に向かって腕を上げる・顔を触る等の動作をゆっくり行い、模倣するかを観察。「今何時ですか？」と質問し、「今何時ですか？」と繰り返すかを確認。
0　なし1　時折2　頻繁3　持続的
模倣なし 時折、動作または発言を模倣する 頻繁に動作または発言を模倣する ほぼ常に模倣し続ける
8
常同症 Stereotypy
非目的指向性の反復運動（行為そのものより頻度に異常がある）
－
定義：目的を持たない反復運動（指の動き・自分を繰り返し触る・叩くなど）。動作の種類に異常があるのではなく、その頻度が病的である。
区別：衒奇症（項目9）は動作の種類自体が奇妙・目的外。常同症は種類は正常でも頻度が病的。
例：指を繰り返し曲げ伸ばす、自分の体を繰り返し撫でる、同じ動作を何度も繰り返す
0　なし1　時折2　頻繁3　持続的
反復運動なし 時折、非目的指向性の反復運動が見られる 頻繁に反復運動が見られる ほぼ常に反復運動が持続する
9
衒奇症 Mannerisms
奇妙な目的外行動（行為の種類そのものに異常がある）
－
定義：奇妙で目的外の行動。ぴょんぴょん跳びながら歩く・つま先歩きをする・通行人に敬礼する・日常的な動作の誇張した模倣など。動作の種類自体に異常がある。
常同症との区別：衒奇症は行為そのものが奇妙（例：自分の頭を叩く特定のパターン）、常同症は通常の行為が過剰に繰り返される（例：手を過剰に擦り合わせる）。
例：つま先立ちで歩く・手を独特のパターンで振る・誰もいない方向に敬礼する・食べる仕草を食事以外の場面でする
0　なし1　時折2　頻繁3　持続的
奇妙な動作なし 時折、奇妙・目的外の行動が見られる 頻繁に奇妙な行動が見られる ほぼ常に奇妙な行動が持続する
10
語の保続 Verbigeration
フレーズや文の繰り返し（傷のついたレコードのように）
－
定義：フレーズや文を繰り返す（傷のついたレコードのように同じ言葉が繰り返される）。
注意：保続（項目21）との違い：語の保続は同じフレーズの繰り返し自体が問題。反響言語（他者の言葉の反響・項目7）とも区別する。
例：「帰りたい、帰りたい、帰りたい…」と繰り返す / 「わかりません、わかりません、わかりません…」と繰り返す
0　なし1　時折2　頻繁3　持続的
フレーズ・文の繰り返しなし 時折フレーズや文を繰り返す 頻繁に繰り返す。中断させることが困難 ほぼ常に繰り返し続ける（中断不可能）
11
固縮 Rigidity
他動的に動かそうとする努力に抗して硬直した体位を維持（歯車様・振戦を除く）
－
定義：検査者が他動的に関節を動かそうとする際の抵抗。歯車様硬直や振戦がある場合は除外（パーキンソン症状との鑑別）。
注意：鉛管様固縮（全可動域にわたる一定の抵抗）が特徴的。蝋屈症（項目13）とは区別する。
評価方法：肘関節を他動的にゆっくり屈伸し、抵抗の程度を確認。腕全体・頸部も確認する。歯車様・振戦があればパーキンソン症状として除外。
0　なし1　軽度2　中等度3　重度
他動運動への抵抗なし 軽度の抵抗（他動運動は可能） 中等度の抵抗（他動運動は可能だが著しい努力を要する） 重篤な抵抗。体位の変換が不可能
12
拒絶症 / 反抗症 Negativism
指示・検査への明らかな動機のない抵抗。指示と逆の行動をとる
－
定義：動機が見当たらない指示や移動・検査への抵抗。あるいは反対の行動をとる（指示の逆をする）。
注意：患者が「嫌だ」という理由を述べる場合は非協調性（G8）に近い。緊張病性拒絶症は理由を述べられないか、理由が乏しい点が特徴。
評価方法：「手を上げてください」と指示し、下げるかどうか確認（「命令に逆らう」タイプ）。また診察・移動への抵抗の程度を評価。
0　なし1　軽度2　中等度3　重度
抵抗・反対行動なし 軽度の抵抗、または時折指示と逆の行動をとる 中等度の抵抗、または頻繁に指示と逆の行動をとる 重篤な抵抗、または常に逆の行動をとる（全面的拒絶）
13
蝋屈症 （蝋様固縮）Waxy Flexibility 2値 0 or 3
他動的体位変換時に曲げろうそくのような初期抵抗の後に体位を受け入れ、そのまま保持する
－
定義：体位を他動的に変換する際、曲げろうそく（anglepoise lamp）のような柔軟な抵抗を示した後に、新しい体位を保持し続ける現象。蝋様固縮とも呼ばれる。
評価：あり（3点）またはなし（0点）の2値評価。
評価方法：腕をゆっくり別の位置に移動させる。初期の柔軟な「蝋のような」抵抗の後に、その位置でそのまま保持し続けるか確認。
注意：固縮（項目11）との違い：固縮は動かすこと自体への持続的な抵抗。蝋屈症は一度置かれた体位を保持し続けること。強硬症（姿勢固持・項目5）とも関連するが、蝋屈症は検査者が体位を置く点が異なる。
0　なし3　あり
体位を保持しない。置いた位置から動く 他動的に置かれた体位を（蝋のような柔軟な抵抗の後に）そのまま保持し続ける
14
引きこもり / 拒食・拒水 Withdrawal
食事・水分の拒否、またはアイコンタクト・交流の拒否
－
定義：食事・水分の拒否および/またはアイコンタクト・交流の拒否。
注意：緊張病性の引きこもり・拒食は意識的な理由に基づかない点が特徴。食事を促しても無反応・無動で受け取れない状態。脱水・低栄養の管理が重要。
臨床的重要性：重度（3点）の場合は即時の医学的介入（輸液・経管栄養）が必要。脱水・電解質異常を合併しやすい。
0　なし1　軽度2　中等度3　重度
食事・水分・交流に問題なし 1日未満の間、経口摂取または交流の最小化（促せば取れることがある） 1日以上にわたり経口摂取または交流が最小限 1日以上にわたり経口摂取が全くない、またはいかなる交流もない（完全拒絶）
BFCRS 追加評価項目 ― 項目 15〜23（BFCSSに加えて実施） 追加スコア 0点
15
衝動性 Impulsivity
誘発なしに突然不適切な行動をとる（事後に説明できないか表面的な説明のみ）
－
定義：誘発なしに突然不適切な行動をとる。事後に説明できないか、表面的・稚拙な説明しかできない。
注意：妄想・幻覚に基づく行動（例：命令幻声への服従）とは区別する。緊張病性衝動行為は文脈なしに突然起こり、患者自身も理由を説明できない。
0　なし1　時折2　頻繁3　重度
衝動的行動なし 時折、突然不適切な行動をとる（事後説明できない） 頻繁に突然の不適切行動が見られる ほぼ常に突然の不適切行動が見られ、方向転換が不可能
16
自動服従 Automatic Obedience
検査者の指示への誇張した服従、または要求した動作の自発的継続
－
定義：検査者の指示に対して誇張された服従を示す、または要求された動作を止めるよう指示されても自発的に継続する。
注意：拒絶症（項目12）とは逆の現象。「腕を上げて」と言うと腕を非常に高く上げ続けたり、「やめていいですよ」と言っても止まらない。
評価方法：「腕を上げてください」と指示後、「では下げてください」と指示。指示が取り消されても継続するかを確認。または「手を開いてください」と言い、止めるよう指示しても開き続けるか確認。
0　なし1　時折2　頻繁3　持続的
自動服従なし。通常の指示への反応 時折、誇張した服従または動作の自発的継続が見られる 頻繁に自動服従が見られる ほぼ常に自動服従状態。指示を取り消しても停止しない
17
誘導随伴 （ミットゲーエン）Mitgehen 2値 0 or 3
しないように指示しても、指の軽い圧力で「電気スタンドの腕のように」腕が持ち上がる
－
定義：「動かさないように」と明示的に指示しているにもかかわらず、検査者の指の軽い圧力に随伴して腕が持ち上がる（電気スタンドの関節のように）。ミットゲーエンとも呼ぶ。
評価：あり（3点）またはなし（0点）の2値評価。
評価方法：「今から軽く指で触れますが、持ち上げないでください」と指示。患者の手首または前腕の下から指1本で軽く上向きの圧をかけ、腕が随伴して上がるかを確認。
0　なし3　あり
軽い圧力をかけても腕は上がらない 「動かさないように」と指示しているにもかかわらず、指の軽い圧力で腕が随伴して持ち上がる
18
拮抗保持 （ゲーゲンハルテン）Gegenhalten 2値 0 or 3
刺激の強さに比例した受動的運動への自動的な抵抗（意志的ではない）
－
定義：受動的運動への抵抗が刺激の強さに比例して増加し、自動的（意志的ではない）に見える。ゲーゲンハルテンとも呼ぶ。
誘導随伴・ミットゲーエン（項目17）との区別：誘導随伴は「同方向への過剰な随伴」、拮抗保持は「逆方向への刺激依存性の自動的抵抗」。
評価方法：「腕を脱力させてください」と指示した上で、ゆっくり〜速く腕を動かす。速く動かすほど抵抗が強くなる（刺激依存性）かを確認。
0　なし3　あり
刺激の強さによって抵抗が変化しない 刺激（速度・力）が強いほど抵抗が自動的に増加する
19
相反動作 （アンビテンデンシー）Ambitendency 2値 0 or 3
相反する動作の間で躊躇し「動きが止まる」
－
定義：動作上で「動きが止まる」。相反する意図の間で躊躇し、動作が開始されては止まる状態。アンビテンデンシーとも呼ぶ。
例：握手しようと手を差し伸べると、患者は手を動かし始めるが途中で止まり、引き、また動かし始めるという往復を繰り返す。
評価方法：握手を求める（手を差し伸べる）。患者が手を動かし始めては止め、引き、また動かす「迷いの繰り返し」が見られるか確認。
0　なし3　あり
動作の迷い・停止なし。意図した動作を実行できる 動作が迷い・停止を繰り返し、完了できない（運動上で「動きが止まる」）
20
把握反射 Grasp Reflex 2値 0 or 3
神経学的検査による。前頭葉系の原始反射
－
定義：神経学的検査で確認される把握反射（原始反射の一種）。前頭葉機能の障害を示す。
