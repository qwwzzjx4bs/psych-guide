STEP 6

モンタージュ理解

「なぜ同じ脳波なのに見え方が違うのか」—双極誘導・基準電極誘導・位相逆転を視覚的に理解する。

モンタージュとは

モンタージュ（Montage）とは、どの電極とどの電極の電位差を記録するかの「組み合わせのルール」です。同じ頭皮電位でも、モンタージュが変わると波形の見え方が大きく変わります。

双極誘導（Bipolar）

隣接する2つの電極の差分を記録。局所性の把握に優れる。波形の振幅は2電極間の電位差を反映。

記録 = 電極A − 電極B

基準電極誘導（Referential）

各電極を共通の基準点と比較。絶対的な電位を把握しやすい。基準に活動があると全誘導に混入する（基準汚染）。

記録 = 電極X − 基準（A1, Cz, 平均等）

主なモンタージュ一覧

縦双極

Fp1-F3-C3-P3-O1 / Fp2-F4-C4-P4-O2（前後方向の隣接差分）。最も広く使われる基本モンタージュ。

横双極

Fp1-Fp2 / F7-F3-Fz-F4-F8（左右方向の隣接差分）。水平ダイポール（BECTS等）の確認に有用。

耳垂基準

各電極 − A1またはA2（耳垂）。耳垂に活動がある場合は基準汚染が生じる。

平均基準

各電極 − 全電極の平均。基準汚染が分散するが、全般性放電では見かけ振幅が低下する。

Cz基準

各電極 − Cz。睡眠段階評価・ICU記録でよく使われる。側頭葉評価に適する。

縦双極モンタージュ 接続図

矢印の方向 = 入力1 − 入力2（正の入力に対して下向きに振れる）

双極誘導（Bipolar Montage）の特徴

長所

  • • 焦点の位置特定に優れる
  • • 基準汚染の影響を受けない
  • • 位相逆転（Phase reversal）で焦点を同定できる

短所

  • • 全般性放電の振幅が見かけ上低下する
  • • 隣接する活動がキャンセルされることがある
  • • 絶対電位を知ることができない

重要原則：双極誘導で「波形が出ない」誘導があっても、その電極の活動が「ゼロ」ではない。隣接する2電極の電位が等しい（差分=0）か、電極の間に焦点があることを意味する。

縦双極モンタージュ — 左後頭部の焦点性棘波

読み方：P3-O1誘導に最大の棘波が出現 → O1が焦点。前後の誘導（C3-P3、O1-O2）でも小さく出現（電場の広がり）。双極誘導では「どの誘導に最大振幅が出るか」で焦点位置を推定する。

基準電極誘導（Referential Montage）の特徴

長所

  • • 各電極の絶対的な活動量を把握しやすい
  • • 全般性放電の把握・振幅比較に適する
  • • 活動の極性（正/負）を判断しやすい

短所（基準汚染）

  • • 基準電極に活動があると全誘導に混入する
  • • 耳垂基準：側頭葉活動が基準に混入
  • • Cz基準：正中構造の活動が基準に混入
耳垂基準誘導 — 同一の左後頭部棘波

読み方：O1に最大振幅が出現し、P3・T5にも小振幅が出現（電場の広がり）。基準（A1）に活動がなければ、活動の大きい電極が焦点に近い。双極誘導と異なり「どの電極が最大振幅か」で直接焦点を読み取れる。

基準汚染の例 — 側頭葉放電が耳垂基準に混入

基準汚染：T3（側頭部）の放電が耳垂（A1）に混入すると、全誘導に逆相の波形が出現する。この場合、双極誘導に切り替えて確認することが重要。

位相逆転（Phase Reversal）—焦点同定の基本技法

双極誘導では、棘波の焦点（電位が最大）を挟む2つの誘導で波形の向きが逆転（位相逆転）します。この「山が下向きになる点」が焦点電極の位置を示します。

原理

焦点の前の誘導

電極A − 焦点B
= 小 − 大 = 負
→ 上向きに振れる

焦点の位置

最大電位の電極
（位相逆転の「谷」）
→ 棘波の焦点

焦点の後の誘導

焦点B − 電極C
= 大 − 小 = 正
→ 下向きに振れる

位相逆転 — 焦点を変えて比較

位相逆転の読み方 3ステップ

  1. 1.最大振幅の誘導を見つける（棘波・鋭波が一番大きい誘導）
  2. 2.その前後の誘導で波形の向きが逆転しているか確認する
  3. 3.逆転の「接点」となる電極が焦点（最大電位点）—それが局所てんかん焦点の位置を示す

同一の焦点性棘波（C3焦点）— モンタージュ別の見え方

縦双極

F3-C3に最大↑
C3-P3に最大↓
（C3で位相逆転）

耳垂基準（A1）

C3に最大振幅
電場の広がりに比例
（直接焦点がわかる）

Cz基準

C3−Czの差分
Czにも活動があれば
振幅が減衰

縦双極モンタージュ（C3焦点）
耳垂基準誘導（A1基準）
Cz基準誘導

比較のまとめ

  • • 双極誘導：C3を挟む「F3-C3」と「C3-P3」で位相逆転 → C3が焦点と特定できる
  • • 耳垂基準：C3誘導が最大振幅 → C3が焦点と直接わかる（ただし耳垂汚染に注意）
  • • Cz基準：CzとC3が近接しているため、C3-Czの差分が小さくなり見逃しやすい
  • • 焦点同定には双極誘導 + 基準電極誘導の両方を確認することが推奨される
