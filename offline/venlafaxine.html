薬理作用・受容体プロファイル

作用機序

ベンラファキシンはセロトニントランスポーター（SERT）およびノルアドレナリントランスポーター（NET）を競合的に阻害し、シナプス間隙の 5-HT・NE 濃度を上昇させる。ムスカリン受容体・ヒスタミン受容体・α1/α2 受容体への親和性は極めて低く（Ki >10,000 nM）、これが SSRI や TCA と比較して抗コリン・鎮静・起立性低血圧などの副作用プロファイルが少ない理由である。活性代謝物 O-デスメチルベンラファキシン（ODV＝デスベンラファキシン） は親薬物より SERT・NET ともにやや強い阻害活性を示す（SERT IC₅₀ ≈ 47 nM、NET IC₅₀ ≈ 531 nM）。

  • • 高血圧（治療中でも血圧モニタリング継続）
  • • 心疾患（頻脈・QTc 延長リスク考慮）
  • • 双極性障害（躁転誘発リスク）
  • • てんかん・痙攣既往
  • • 緑内障（眼圧上昇リスク）
  • • 出血傾向・抗凝固薬使用中（特にNSAIDs）
  • • 高齢者（転倒・低 Na 血症リスク）
  • • 自殺念慮・既往（特に投与初期）

SSRI → ベンラファキシン 切り替えプロトコル

クロステーパー法（推奨）
1
現在の SSRI を 50% に減量しながらベンラファキシン 37.5 mg/日 を開始
2
1–2 週後：SSRI を 25% まで漸減、ベンラファキシン 75 mg/日 へ増量
3
さらに1–2 週後：SSRI 中止、ベンラファキシン維持用量（75–150 mg）
注意事項
  • • セロトニン症候群の症状に注意（発熱・ミオクローヌス・過反射）
  • • フルオキセチン（t₁/₂ 長い）は CYP2D6 強力阻害 → ベンラファキシン AUC 大幅上昇リスク
  • • 切り替え速度は患者耐容性に応じて個別化
  • • 離脱症状（前薬）と新薬副作用を区別

ベンラファキシンを優先検討する状況

  • • SSRI が 2 剤以上無効の難治性うつ病
  • • うつ病に疼痛症状（身体化・慢性痛）が合併
  • • GAD と MDD の合併（単剤で両疾患に対応）
  • • 性機能障害が少ない SSRI を求めるが効果不十分な場合
  • • 倦怠感・集中力低下が前景にある MDD（NE 作用への期待）

SSRI を優先する状況（ベンラファキシン選択に慎重）

  • • 高血圧がコントロール不良
  • • 中止症候群リスクが高い（アドヒアランス不良が予測される）
  • • 双極性障害が疑われる（躁転誘発）
  • • 妊娠中・妊娠希望（第一選択は SSRI）
  • • 安価なジェネリック SSRI が有効な場合

具体的な適用症例

以下の症例は架空であり、教育目的のみに使用します。実際の患者への適用は最新の添付文書・ガイドラインに従ってください。

症例 A うつ病 — SSRI 2剤無効後
患者：48歳男性・会社員、中等度うつ病（PHQ-9: 15）
既往：エスシタロプラム 20mg（8週）・パロキセチン 40mg（8週）ともに効果不十分
ベンラファキシン選択の理由
  • • SSRI とは異なる作用（NET 追加）による反応期待
  • • 倦怠感・集中力低下が主訴 → NE 効果に期待
  • • 疼痛（肩こり・頭痛）合併 → SNRI が適切
方針：37.5mg 開始→4週後 150mg へ段階増量。12週後 PHQ-9: 5（寛解）。
症例 B 社会不安障害＋うつ状態の合併
患者：32歳女性・会社員、SAD（LSAS ≥60）+ 軽度うつ状態（PHQ-9: 9）
既往：SSRIに対し消化器副作用で中止歴1回
ベンラファキシン選択の理由
  • • SAD とうつ病を単剤でカバー（海外エビデンス豊富）
  • • タンパク結合率が低い（DDI リスク低い）
  • • 食欲増進・体重増加が少ない
方針：37.5mg 開始→75mg 維持。8週後 LSAS 改善、PHQ-9: 4（寛解）。
症例 C 慢性腰痛合併うつ（疼痛・うつ併存）
患者：56歳男性、慢性腰痛（VAS 6/10）+ うつ病（PHQ-9: 13）
既往：整形外科治療中、NSAIDs使用。うつ病未治療。
ベンラファキシン選択の理由
  • • NE 経路による下行性疼痛抑制機序
  • • 疼痛＋抑うつを単剤で対処（デュロキセチンとの比較考慮後）
  • • 疼痛軽減には 150 mg 以上が必要な場合あり
方針：37.5→75→150mg へ増量。NSAIDs との相互作用（出血リスク）に注意。12週後 VAS: 3/10、PHQ-9: 6。
症例 D 難治性うつ — 高用量（225 mg）使用
患者：52歳女性、反復性うつ病（F33）、第4エピソード。PHQ-9: 20（重度）
既往：SSRI 2剤・ベンラファキシン 75mg（各12週）いずれも部分反応のみ
高用量 225mg 使用の根拠
  • • 75mg では NET 占有不十分 → 150–225mg で真の SNRI 効果
  • • 血圧モニタリング強化（週1回 → 月2回）
  • • リチウム増強療法を同時に検討
方針：75mg から75mg 単位で225mgまで増量（各2週間）。血圧+8/6mmHg上昇 → 降圧薬調整。16週後 PHQ-9: 9（奏効）。

最新研究動向（2020–2026年）

RCT 2023年

ボルチオキセチン vs ベンラファキシン — 認知機能比較 RCT

成人 MDD 患者を対象に認知機能（RAVLT, TMT-A/B 等）を主評価として比較。ボルチオキセチン群でいくつかの認知指標で優越性を示した一方、ベンラファキシンも PHQ-9 改善では同等。認知機能改善が前景の場合はボルチオキセチンを優先する根拠となりうる。

RCT 2024年

難治性うつ（TRD）― 増強 vs 切り替え比較（ASCERTAIN-TRD）

SSRI/SNRI 不応の TRD を対象に、アリピプラゾール増強療法 vs SNRIへの切り替え（ベンラファキシン XR / デュロキセチン）を比較。増強・切り替えともに一定の有効性。「増強か切り替えか」のアルゴリズム上の選択を検証した重要試験。

Pharmacogenomics 2023–2024年

CYP2D6 表現型と臨床アウトカムの関係 — 遺伝薬理学的エビデンス

高齢者 PK モデル（2024, PubMed 38284409）および系統的レビューにより、CYP2D6 PM 表現型では VEN AUC が大幅に上昇するが活性モイエティ（VEN + ODV）は EM と類似することが再確認。現時点では CPIC は VEN への必須ゲノタイプ検査を推奨していないが、強い CYP2D6 阻害薬を併用する場合は注意が必要。

Meta-analysis 2021–2022年

デュロキセチン vs ベンラファキシン — ネットワークメタ解析

直接比較試験が少ないが、ネットワークメタ解析ではベンラファキシンが MDD 短期治療でわずかに効果量が高い傾向が示されている。一方デュロキセチンは疼痛適応（線維筋痛症・DPN 等）で日本での承認が広い。両薬の差は臨床的に小さく、適応症・副作用プロファイルで使い分けが現実的。
