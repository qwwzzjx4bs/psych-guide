精神保健福祉法の体系

昭和25年 法律第123号

法の目的（第1条）

精神障害者の権利擁護を図りつつ、医療・保護を行い、社会復帰の促進と自立を支援し、精神的健康の保持・増進に努めることで、国民の精神保健の向上を図る。

精神保健指定医とは

法第18条
制度創設の経緯

1983年（昭和58年）の宇都宮病院事件（看護職員の暴行によって患者が死亡）を契機に、昭和62年の法改正で創設された。人権上配慮を要する精神医療行為に、一定の資質を有する医師の判断を必須とする制度。

制度の目的

精神保健指定医は、患者の人権に配慮した適切な精神医療を提供することを求められる。入退院判定・行動制限など患者の自由を制限する行為について、法的権限を付与される唯一の資格。

令和4年精神保健福祉法改正のポイント

R4.12.16公布

令和4年改正は複数の施行日に分かれています。特に令和6年4月1日施行の内容は、申請するケースレポートに反映が必要です（R6年4月以降入院の症例を提出する場合）。

令和4年12月16日施行

目的規定における権利擁護の明確化

法第1条（目的規定）に「精神障害者の権利の擁護を図りつつ」という文言が明確化された。精神医療が人権擁護を中心に据えることを宣言。

令和5年4月1日施行

入院患者への告知の拡充

患者本人だけでなく家族等にも告知。「入院措置を採ること」「退院請求」に加えて「入院理由」も告知義務化。

研修会の有効期間延長

指定医研修会受講後の申請可能期間が「1年以内」→「3年以内」に延長（R5年4月1日以降申請から適用）。

虐待加害者を「家族等」から除外

DV・虐待の加害者は医療保護入院の同意主体となる「家族等」から除外。当該家族が唯一の場合は市町村長同意可。

令和6年4月1日施行 ケースレポートに影響あり

医療保護入院の入院期間の法定化 ★最重要

  • • 入院から6か月未満：最大3か月以内で入院期間を定める
  • • 入院から6か月以降：6か月以内ごとに更新
  • • 更新要件：①指定医診察で必要性確認 ②退院支援委員会で審議 ③家族等の同意（または市町村長同意）
  • • 更新届を都道府県等に提出（定期病状報告は廃止）

その他の主な改正（R6年4月〜）

  • • 市町村長同意の拡大：家族等が全員「同意・不同意の意思表示を行わない場合」も市町村長同意可（法第33条第2項）
  • • 入院者訪問支援事業の創設（法第35条の2）：都道府県等が訪問支援員を派遣し、入院者の傾聴・相談支援を実施
  • • 虐待防止措置の義務化：精神科病院管理者に研修・相談体制整備を義務付け。業務従事者による虐待発見者の通報義務
  • • 措置入院時の精神医療審査会審査：措置入院時にも審査会が入院の必要性を審査
  • • 退院後生活環境相談員の措置入院者への選任義務化
【発展】医療保護入院の「みなし同意」とは

入院期間更新時に家族等へ通知を行った後、厚生労働省令で定める日（通知から2週間経過した日）までに不同意の意思表示がなかった場合、同意を得たものとみなすことができる。ただし以下の場合はみなし同意不可：①定期的な連絡が行われていない、②通知後に該当家族が死亡・意思表示不能になった、③別の家族等に通知した、④2週間経過日が入院期間満了日を超える。

行動制限最小化

法第36条第3項

なぜ行動制限を最小化しなければならないか

人権（尊厳）
基本的人権「人身の自由」を制限する行為であり、極めて慎重な対応が必要
身体的影響
深部静脈血栓症・肺塞栓症による死亡例、関節拘縮・褥瘡の発生
心理的影響
トラウマ・医療従事者への陰性感情・医療者自身の士気の低下

行動制限（隔離・身体拘束）の判断の3要件

①切迫性

本人または他者の生命・身体が危険にさらされる可能性が著しく高い

②非代替性

隔離・身体拘束以外に代替する手段がない

③一時性

隔離・身体拘束は一時的なものであること

絶対禁止（法第36条第2項・昭和63年厚生省告示第128号）：信書の発受の制限 / 都道府県・人権擁護機関職員との電話・面会の制限 / 弁護士・患者代理人となろうとする弁護士との面会の制限 → 指定医であっても禁止できない

隔離（昭和63年厚生省告示第129号）

「内側から患者本人の意思によっては出ることができない部屋に1人だけ入室させることにより他の患者から遮断すること」で12時間を超えるものに限る

対象となる患者

  • ア）他の患者との人間関係を著しく損なうおそれ等
  • イ）自殺企図・自傷行為が切迫
  • ウ）他の患者への暴力・迷惑行為・器物破損
  • エ）急性精神運動興奮で一般精神病室での医療が困難
  • オ）身体合併症で検査・処置等のため隔離が必要

遵守事項

  • • 同一隔離室に複数患者を入室させてはならない
  • • 隔離を行う理由を患者に知らせるよう努める
  • • 診療録に開始・解除日時と理由を記載
  • • 原則として少なくとも毎日1回医師が診察
  • • 衛生の確保（洗面・入浴・掃除等）

身体的拘束（昭和63年厚生省告示第130号）

「衣類または綿入り帯等を使用して、一時的に患者の身体を拘束し、その運動を抑制すること」。隔離より制限の程度が強い

対象となる患者（より厳格）

  • ア）自殺企図・自傷行為が著しく切迫
  • イ）多動または不穏が顕著
  • ウ）そのまま放置すれば生命にまで危険が及ぶおそれ

遵守事項

  • • 専用に配慮された衣類・綿入り帯等を使用
  • • 手錠・縄・紐等は絶対に使用禁止
  • • 診療録に開始・解除日時と理由を記載
  • • 原則として常時の臨床的観察
  • • 医師は頻回に診察（漫然と継続しない）

令和3年最高裁決定（口頭試問で問われる可能性あり）

医療保護入院中の患者への身体的拘束に関する損害賠償請求において、患者側勝訴の控訴審判決（名古屋高裁金沢支部 令和2年12月16日）を最高裁が不受理（令和3年10月19日）として確定。ポイント：「看護師8名で対応すると患者が不穏にならなかった経験があるのに、その都度大人数での対応ができないとして身体拘束を継続したことは、非代替性の要件を満たさない」と判示。

変更・紛失等の届出
  • • 指定医証の記載事項の変更（氏名・所属機関等）
  • • 住所地の変更は必ず届け出ること（研修案内不達防止）
  • • 紛失・き損時も届出が必要
  • • 届出先：都道府県知事または指定都市の長
指定の取消し・停止（法第19条の2）

必ず取り消される場合（第1項）

  • • 医師免許が取り消された場合
  • • 期間を定めて医業の停止を命じられた場合

取り消せる場合（第2項）

  • • 精神保健福祉法もしくはその命令に違反
  • • 職務に関し著しく不当な行為（例：不当な身体拘束、患者財産横領等）
  • • 著しく不適当と認められるとき
  • • ※ 処分前に医道審議会の意見を聴く必要あり

失効後1年未満の特例申請

指定失効後1年未満の者に限り、更新研修受講後にケースレポート等の必要書類を一部省略して新規申請が可能。ただし資格は失効しているため、再指定を受けるまで指定医業務は不可。
